# Student Management System

A simple full-stack Student Management System built using HTML, JavaScript, Express.js, and MongoDB. The application allows users to manage student records through a clean and user-friendly interface.

## Technologies Used

- HTML
- CSS
- JavaScript
- Express.js
- MongoDB
- Mongoose

## Features

- Add new students
- View all students
- Edit student details
- Delete students
- Search students by name, roll number, or department
- Store student information (Name, Roll Number, Department, Email)
- Input validation
- Student counter
- Delete confirmation



## How to Run

1. Clone the repository

```bash
git clone <your-repository-url>
```

2. Install dependencies

```bash
npm install
```

3. Make sure MongoDB is running.

4. Start the server

```bash
node server.js
```

5. Open your browser and visit:

```
http://localhost:3000
```
